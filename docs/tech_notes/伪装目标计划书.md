## 1\. 项目综述与环境准备

本项目旨在基于 **HitNet (High-resolution Iterative Feedback Network)** 架构完成伪装目标检测（COD）任务。项目利用 Transformer (PVT-v2) 提取特征，并通过迭代反馈机制细化边缘，以解决多尺度和跨域检测难题。

基于 **HitNet (High-resolution Iterative Feedback Network)** 架构，完成“未来杯”2025赛题A的三项任务：

1.  **基础检测 (Q1):** 在CAMO数据集上训练，输出掩码与评估指标。
2.  **多尺度检测 (Q2):** 实现对256\~2048不同分辨率输入的自适应检测。
3.  **泛化与跨域 (Q3):** 在Camouflage-people及NC4K上测试，并输出检测框。

### 2.2 数据加载器 (Dataset Implementation)

鉴于 `CAMO/CAMO-D/gt` 包含掩码图，我们优先使用图像加载，同时保留 JSON 解析功能以适配 NC4K 测试集。

**文件路径**: `src/dataset.py`

```python
import os
import cv2
import torch
import numpy as np
from torch.utils.data import Dataset
from pycocotools.coco import COCO
import albumentations as A
from albumentations.pytorch import ToTensorV2

class CODDataset(Dataset):
    def __init__(self, root_dir, json_file=None, image_dir=None, mask_dir=None, mode='train', size=352):
        """
        mode: 'train' (开启增强), 'val', 'test'
        对于CAMO训练：传入 image_dir 和 mask_dir (直接读取GT图，速度快)
        对于NC4K测试：传入 json_file (需从JSON解析ID)
        """
        self.mode = mode
        self.size = size
        self.use_json = json_file is not None

        if self.use_json:
            self.coco = COCO(json_file)
            self.ids = list(self.coco.imgs.keys())
            self.root_dir = root_dir # 图片根目录
        else:
            self.image_paths = [os.path.join(image_dir, f) for f in os.listdir(image_dir) if f.endswith('.jpg')]
            self.mask_dir = mask_dir

        # 训练集强增强 (针对问题三的泛化能力)
        self.transform_train = A.Compose([
            A.Resize(size, size),
            A.HorizontalFlip(p=0.5),
            A.VerticalFlip(p=0.5),
            A.RandomRotate90(p=0.5),
            # 模拟光照变化和噪声，应对 Camouflage-people 的人工伪装
            A.RandomBrightnessContrast(p=0.2),
            A.GaussNoise(p=0.2),
            A.Normalize(mean=(0.485, 0.456, 0.406), std=(0.229, 0.224, 0.225)),
            ToTensorV2(),
        ])

        self.transform_val = A.Compose([
            A.Resize(size, size),
            A.Normalize(mean=(0.485, 0.456, 0.406), std=(0.229, 0.224, 0.225)),
            ToTensorV2(),
        ])

    def __len__(self):
        return len(self.ids) if self.use_json else len(self.image_paths)

    def __getitem__(self, index):
        # 1. 读取图像和掩码
        if self.use_json:
            img_id = self.ids[index]
            img_meta = self.coco.loadImgs(img_id)[0]
            image_path = os.path.join(self.root_dir, img_meta['file_name'])
            image = cv2.imread(image_path)
            image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
            filename = img_meta['file_name']

            # 如果是测试集，可能没有mask
            mask = np.zeros(image.shape[:2], dtype=np.uint8)
        else:
            image_path = self.image_paths[index]
            filename = os.path.basename(image_path)
            image = cv2.imread(image_path)
            image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)

            # 读取对应的GT掩码
            mask_name = filename.replace('.jpg', '.png') # 假设gt是png格式
            mask_path = os.path.join(self.mask_dir, mask_name)
            if os.path.exists(mask_path):
                mask = cv2.imread(mask_path, 0) # 读取灰度
                _, mask = cv2.threshold(mask, 127, 1, cv2.THRESH_BINARY)
            else:
                mask = np.zeros(image.shape[:2], dtype=np.uint8)

        # 2. 应用增强
        if self.mode == 'train':
            augmented = self.transform_train(image=image, mask=mask)
        else:
            augmented = self.transform_val(image=image, mask=mask)

        img_tensor = augmented['image']
        mask_tensor = augmented['mask'].float().unsqueeze(0) # [1, H, W]

        return img_tensor, mask_tensor, filename
```

**关键代码逻辑 (`train.py`)**:

2.  **多尺度监督**：HitNet 输出 4 个尺度的结果，需要对每个尺度计算 Loss 并相加。
    ```python
    loss = loss_func(pred_1, mask) + loss_func(pred_2, mask) + ...
    ```
3.  **评价指标计算**：
    在验证循环中，计算 MAE 和 IoU。
    ```python
    def calculate_metrics(pred, gt):
        # pred, gt 均为 Tensor [B, 1, H, W]
        intersection = (pred * gt).sum()
        union = pred.sum() + gt.sum() - intersection
        iou = intersection / (union + 1e-6)
        mae = torch.abs(pred - gt).mean()
        return iou, mae
    ```

### 3.3 问题三：泛化与检测框转换 (Task 3)

**目标**：NC4K (生成 BBox) 和 Camouflage-people (跨域适应)。

**泛化策略**：

  * 在 `Dataset` 中我们已经加入了 `ColorJitter`, `GaussNoise` 等增强，这能有效提升对人工伪装（Camouflage-people）的识别率。
  * **测试时增强 (TTA)**: 在推理 NC4K 时，对图像进行水平翻转预测，然后将结果与原图预测取平均。

**掩码转检测框逻辑 (针对 NC4K)**:
由于 HitNet 输出的是分割掩码，需转换为 Bounding Box。

```python
def mask_to_bbox(mask_binary):
    # mask_binary: 0/1 numpy array
    contours, _ = cv2.findContours(mask_binary, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    bboxes = []
    for cnt in contours:
        x, y, w, h = cv2.boundingRect(cnt)
        if w * h > 50: # 过滤噪点
            bboxes.append([x, y, x+w, y+h])
    return bboxes
```

  * **可视化**：对于问题三要求的可视化图像，使用 `cv2.rectangle` 将生成的框画在原图上。
